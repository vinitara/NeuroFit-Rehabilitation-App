## App Overview
NeuroFit is a rehabilitation app aimed to provide _**accessible physical theraphy for patients of stroke**_. With NeuroFit, physical therapy can 
be done anywhere and anytime, using an iPad, revolutionizing the way physical therapy has been done, making it simple, sustainable, and 
suitable for both the patients and their families. By using AI technology, the application can detect and correct any inaccuracies in the 
patient's posture, making physical therapy more effective and reducing the workload for physical therapists. Furthermore with NeuroFit, 
physical therapy is no longer a tedious task but rather an engaging and fun experience that can be done from the comfort of one's own home.

## App Development
*Swift language* is used to develop the app using *Swift Playground* and *XCode* as the main development program for this app
