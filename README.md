## App Overview
NeuroFit is a rehabilitation app aimed to provide _**accessible physical theraphy for patients of stroke**_. With NeuroFit, physical therapy can 
be done anywhere and anytime, using an iPad, revolutionizing the way physical therapy has been done, making it simple, sustainable, and 
suitable for both the patients and their families. 

By using _**AI technology**_, the application can detect and correct any inaccuracies in the 
patient's posture, making physical therapy more effective and reducing the workload for physical therapists. Furthermore with NeuroFit, 
_**physical therapy is no longer a tedious task but rather an engaging and fun experience**_ that can be done from the comfort of one's own home.
 
## App Development
*Swift language* is used to develop the app using *Swift Playground* and *XCode* as the main development program for this app
NeuroFit was developed using _**Swift**_, with _**Swift Playgrounds**_ and _**Xcode**_ as the main development tools.

## Features Overview
### Exercise Page
#### Exercise Information 
* Displays exercise duration and target area
* Explains the benefits of each exercise
* Provides simple step-by-step instructions
* Includes _**voice-to-text narration**_ description for exercise desciption and instruction
#### Camera View
* Uses an _**AI Joint detector**_ to monitor user's movements
* Gives in real time feedbacl when exercises are performed inccorectly (ex."Raise your arm")
* Automatically counts repetitions based on detected movements
* Includes a timer, pause button, and end session button
---
### Daily Challenge Page
* Both games includes a simpler version of the game to aid users with different abilities
#### Memory Game
* Users are shown a set of emojis before they are hidden. The goal is to match pairs of identical emojis
* Helps *improve user's short-term memory, focus and cognitive function*
#### 15-Puzzle Game
* Users must slide numbered tiles to arrange them in order from 1–15 on a 4×4 grid.
* Helps *improve user's problem-solving and planning skills*
---
### Calendar Page
*  Keeps track of completed exercise sessions
* Allows users to review their rehabilitation progress over time
* Includes a symptom log for recording recovery related observations
___
### Profile Page
* Stores user information
* Includes an automatic BMI caculator based on the user's weight, height and age
___
## Game Aspect
* The app includes a _**badge and achievement system**_ that rewards users for reaching rehabilitation milestones such as Completing their first exercise session, Exercising for 3 consecutive days, ect
* This system helps _**motivate users to stay consistent with their rehabilitation**_ and encourages long-term engagement with daily exercises.
